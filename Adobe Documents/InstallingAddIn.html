Adobe Connect: Install the Add-In

When people are experiencing difficulties with audio or video in Adobe Connect, but they have all their devices connected, their problems are most likely to be solved by doing a one-time installation of the Adobe Connect Add-In. Here's how.

Install from outside a meeting

From outside a meeting (i.e., before launching an actual meeting):

1.   browse to http://www.adobe.com/support/connect/downloads-updates.html (Links to an external site.)

2.   choose the Adobe Connect Meeting Add-In for your operating system. We are currently using Adobe Connect 9.x, so choose the Add-In for version 9.

Install from within a meeting

From within a meeting running version 9, Connect will generally prompt you to install the Add-in. If not, either:

1.   browse to the Meeting Menu on the top left

2.   choose Audio Setup Wizard

3.   click the button to Install Add-In

4.   After the install Connect will close your session in the browser and automatically start the session in the Add-In

Or:

1.   browse to the Help menu on the top right

2.   Click on the Downloads link

3.   Find the Connect Add-In for your operating system and click the link

4.   Follow the instructions to complete the install.

5.   After the install Connect will close your session in the browser and automatically start the session in the Add-In

 

Note: The Connect Add-In will not run if you are trying to access the meeting room from the Chrome web browser. Do not use Chrome to start or enter an Adobe Connect Pro session, especially if you are expected to be a presenter.
