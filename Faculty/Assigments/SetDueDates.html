How to Set Assignment Due Dates

Materials:

This procedure assumes that an assignment is already developed and only explains how to set due dates for that assignment.


Procedure:

1.   From the Course Home Page, first click on the Turn Editing On button in the upper right corner of the window.

 

 

2.   Click the dropdown arrow located to the right of the assignment you wish to set due dates for.

 

 

 

 

 

3.   On the Assignment Settings page scroll down just past the description field and find the Availability Settings section and adjust the following fields:

 

 

a.    Allow submissions from: If enabled, students will not be able to submit before this date. If disabled, students will be able to start submitting right away.
--> Check the Enable checkbox and then set the appropriate date and time.

b. Due Date: This is when the assignment is due. Submissions will still be allowed after this date but any assignments submitted after this date are marked as late. To prevent submissions after a certain date - set the assignment cut-off date.
--> Check the Enable checkbox and then set the appropriate date and time.

c. Cut-off date: If set, the assignment will not accept submissions after this date without an extension.
--> Check the Enable checkbox and then set the appropriate date and time.

 

4.    Scroll to the bottom of the page and click on Save Changes.

NOTE: Do not confuse assignment due dates with the "Restrict Access" options towards the bottom of the page. The difference between the “Restrict Access” from/to dates and assignment availability settings for the activity is that outside the set dates the later (assignment settings) allows students to view the activity description, whereas access from/to dates prevent access completely.

 

5.   Repeat steps 2 – 4 for all assignments throughout the course.
