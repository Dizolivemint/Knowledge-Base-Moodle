Import Documents from a Previous Course

 

 

The purpose of this procedure is to explain the process of importing documents from a previous class directly into a new upcoming class.  For example:  Copy handouts used and located in the Spring term course over to the Fall term course shell to be used in that class.

 

If you prefer to watch Video Tutorials on the process:

 

 Video Tutorial: How to import selected content from a previous class (not including the gradebook)

Video Tutorial: How to import all the content from a previous class (including the gradebook)

 

 

Note:  Master Course Shells – if your course is copied from a master course shell and you would like to have certain documents (ie: handouts, additional resources, etc.) added to the master shell version of your course.  Please create a support request by clicking on the Pacific College eLearning Support link to get in contact with the eLearning development team. Adding documents to the “master course” will ensure those documents are automatically included in each new course session in subsequent semesters.

 

 

 

Materials:

 

To complete this procedure, you will need the following information:

·      The course number of the class to pull documents from (ex. PSY101.DE.S13-SD).

·      The course number of the new class to import documents to.

·      The visual location of the documents or folders in the course that are to be imported.

 

Import documents from a previous course:

 

1.   From the My Courses page (the first page that opens after logging into the eLearning system), click on the course title to open the new course you want to import documents to.

2.   The Course Home Page for the new course will display.  Select Settings, then Import.

 

 

 

3.   Find a course to import data from:  Type the course number you want to pull documents from into the search field at the bottom of the screen (ex:  PSY101), click Search.

 

 

 

4.   Click the button to select the appropriate course listed in the Course Short Name column.  Click Continue.

 

 

 

 

5.   Initial Settings:  The initial settings can be left at their default settings.  No changes need to be made.  Click Next.

6.   Schema Settings:  This page displays all of the content that is available to be imported.  The items that you want to import will be selected on this page.

a.   First, click on the word “None” at the top of the list.  This will uncheck all items.

b.   Locate the folder or documents that you wish to import.  Note:  If you want to import the Shared Files folder you will select the folder not individual files within the folder.  All folder contents will be imported and copied over.

c.   Check the box on the Section Header that contains your documents to activate that section and its contents.

d.   Next, check the boxes by the items you wish to import.  You can select as many as you wish and import them all at the same time.

e.   Scroll to the bottom of the page and click Next.

 

 

 

 

 

7.   Confirmation and Review: The same list of all course items will display, however, a green check will be next to the items scheduled to import and a red x will be next to items that will not be imported. Scroll to the bottom of the page and click Perform Import. This could take up to a few minutes depending how much is being imported.

.

8.   A message in green text will display indicating the import is complete.  Click Continue.

 

 

 

 

9.   The Course Home Page will display; you should now see the newly imported document or folder located at the bottom of the section list.

 

10. Open the folder or document that was just imported to verify that all of the contents are present and it is working properly.  If you experience any problems please click on the Pacific College Support link to submit a request for support.

 

With editing turned on you may click and hold the four-pointed arrow then drag the item to a new location in the list, if desired.
