Upload a Syllabus for a Course

 

 

 

 

Purpose:

 

 

The purpose of this procedure is to explain the process for uploading the course syllabus into athe eLearning  course  shell.   Loading  the course  syllabus  to the eLearning  platform  will give every student  easy access  to the syllabus  document  24-hours  a day, 7-days  a week throughout  the entire semester.

 

 

This procedure covers the following topics:

á      Upload a syllabus for a course.

á      Move the location of the syllabus on the course home page.

 

 

 

 

 

Materials:

 

 

To complete this procedure, you will need the following information:

á      An approved, final draft of your course syllabus saved as a Microsoft Word document.

 

 

 

 

Procedure:

 

 

Upload a Syllabus for a Course

 

 

The syllabus  may be added  to the Shared  Files folder,  however,  for more visibility  and easier access, creating a separate link on the Course Home Page for the syllabus is recommended.

 

 

1.   From the Course Home Page, click on the Turn  editing on button located in the upper right corner of the page.

 

 

 

2.   Locate  and  click  on  the  Add   an  Activity  or  Resource  link  to  open  the  Activity  and

Resource Menu.  Select File (Illustration on next page) and then click Add.

 

 

    

 

 

 

 

 

 

 

3.   On the Adding a new file page, complete the following fields:

a.   Name: Syllabus.  (This is a required field.)

b.  Description:  enter ÒSyllabusÓ again as the description.  (This is a required field, however, it is not visible to anyone if the ÒDisplay description on Course PageÓ box is unchecked.)

c.   Display description on course page:  unchecked.

 

 

 

 

4.   In the Content > Select Files section, click the AddÉ  link.  Note:  you may also drag files from your desktop/Finder/Windows  Explorer  window  and drop them into the drag-n-drop field in lieu of steps 4-11.

 

    

 

 

 

 

5.   In the File Picker window, select Upload a file in the list on the left, if not already selected.

 

 

 

 

 

 

 

6.   Click the Choose  File button after Attachment: in the center of the File Picker window.

 

 

 

 

 

 

 

 

7.   Browse to the location where the Syllabus is located.   Select the file and click Open.

 

 

8.   Save  As:   select a descriptive title for the document so you may recognize it later on (for example:  PSY101_F13_Syllabus.)

 

 

9.   Author: this field will default to your name.

 

 

10. Choose  license: No changes are necessary, leave this setting at All rights reserved.

11. Click the Upload this file button.

 

a.   Depending on the size of the file, it may take several seconds for the file to appear in the Shared Files window.

b.   The Word document icon will display in the Select Files window.

 

 

12. Scroll to the bottom of the page and click on Save and return to course.

 

 

13. On the Course Home Page, the last item listed will now be the Syllabus you just uploaded.

 

 

 

 

 

Note:    The Syllabus  is a Microsoft  Word document  and will not open directly in the eLearning interface.   Depending on your computerÕs Internet browser settings, clicking on the Syllabus link will either initiate an immediate download of the file to your computer, or a pop-up window may

open asking if you would like to open the file (in Microsoft Word) or save the file to your computer.

 

 

Procedure complete

    

 

 

Move the location of the Syllabus on the Course  Home Page

 

 

ItÕs easy to move items around on the Course Home Page for better organization.

 

 

 

1.   From the Course Home Page, click on the Turn Editing On button.

 

 

 

 

 

2.   Click and hold on the four-pointed arrow  (third editing icon from the left).

 

 

 

 

 

3.   With the left mouse button pressed, drag the item towards the top of the page and drop it in its new location between Announcements and Shared Files.

 

a.   Note:   Dropping the file in the exact location can be a bit finicky, and it may take a couple of tries to get it right.

 

 

 

4.   When the item is placed in the right spot, click on the Turn  Editing Off  button.                                                                              The

Syllabus will now be in this new home until it is moved again.

 

 

 

 

 

Procedure complete
