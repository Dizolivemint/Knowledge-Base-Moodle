Adding a New Rubric from Scratch

 

This tutorial will teach how to create and adding a new grading rubric to the eLearning gradebook.

 

1.  Open the assignment or forum page.

Click on the assignment or discussion forum that you wish to add a rubric to (you do not need to have editing turned on). The assignment/forum page will display.

 

2.   Open the Settings menu, select Advanced grading.

You will notice a sub-menu below the Advanced grading link called "Define rubric" that will open, but just click on the Advanced grading link to open the Advanced Grading: <Assignment> (Submissions) page.

 

3.  Select Rubric from the Grading Method drop down menu.

At the top of the page, there is a drop down menu to select a grading method. If you have made no changes on the Assignment Settings page in the "Grade" section, the grading method will likely be "Simple direct method." Select Rubric from the drop down menu to start setting up the assignment to be graded from a rubric.

 

4.  Click the box labeled "Define new grading form from scratch."

The Define rubric settings page will display.

 

5.  Enter a Title and Description for the rubric

IMPORTANT: All rubric templates are listed on one single scrolling page for all classes. Without a unique title, you will never be able to find your rubric template if you attempt to attach this rubric to any other assignments or forums.

Name your rubric in the following manner: Course code Assignment Title. For example ENG101_Final_Paper_Rubric

In the description, please enter your name as the author of the rubric, the class and the date the rubric was uploaded. For example: Written by Jane Smith, English 101, uploaded 04/22/2014.

 

6.  Add criterion and point levels

Enter the criterion, then enter the point rating descriptions (the requirements to achieve that level of points). Simply click inside the box and it will turn into a textbox that you can either type in text, or copy it from a Word document and paste it in (recommended method).

Click the "Add Level" button to add more boxes for additional levels.

Click the "Add criterion" button to add more boxes for additional criterion.

 

 

7.  Enter the point values for each level.

Click on the green points to open a text field to enter the point value for each level. The default is to start with zero on the left side and work its way up, but you can enter your levels either ascending or descending. Just be sure that you enter the correct point value to the appropriate description. See the image to the right for reference.

 

8.  Review the Rubric Options

Check or uncheck options as you see fit. The default is to have all options checked.

 

9.  Click "Save rubric and make it ready"

This will save your rubric and take you back to the Advanced Grading: <Assignment> (Submissions) page.

You will notice that your new rubric is now displayed and next to the title, there is a green box that says "Ready for usage." This indicates that you have successfully added the rubric to the assignment or forum and it is ready to be used to grade papers.

The process above added the rubric to only the assignment that you originally opened. To add this same rubric to other assignments or forums, you will need to open each assignment or forum and add the rubric.

On the same Advanced Grading <Assignment> (Submissions) page. Click the box labeled "Publish the form as a new template." Then click Continue to confirm that you want to copy the rubric to a template. Please note that others who use the Pacific College eLearning system will also have access to the rubric template.

The Advanced Grading <Assignment> (Submissions) page will again display, but with a light blue banner at the top indicating "The form was successfully saved as a template"

The process to load a rubric from a template is written in a separate tutorial.

 

This concludes our tutorial on adding a new grading rubric from scratch. If you have any problems or notice any steps that are incorrect or need tweaking, please submit a support request to the IT Help Desk with detailed descriptions of your specific issue or what needs to be addressed in this tutorial.
