Quizzes: Delete Attempts, Extend Dates/Time, Allow Retakes for Individual Students

 

 

Online quizzes include the flexibility to grant exceptions for special circumstances that may come up. Such as allowing a student additional time to take a quiz, extending the dates the quiz is accessible, or allowing specific students additional attempts to take a quiz. All of these options are available to the instructor and give the instructor authority to make these decisions on their own.

NOTE: All of the options below except deleting an attempt (which is not recommended) give you the opportunity to require a password to access the quiz. Some operating system/browser combinations are autofilling the password field. Be sure the password field is blank before saving the override.

 

 

 

1.   Delete the quiz attempt, if necessary

 

 

IMPORTANT NOTE: The general default quiz setting is to allow only one attempt to complete a quiz. Deleting an attempt should be used ONLY AS A LAST RESORT. You may just need to add a user override for the quiz to allow additional attempts (item 4 below).

To delete a quiz attempt, start by opening the specific quiz you want to work with.
1. With the Quiz page open, select the "Attempts '#'" link (the number will reflect how many students have attempted the quiz).
2. Locate the student, check the box to select their quiz attempt,
3. Scroll to the bottom and click the box to delete selected attempts.

 

 

 

2.   Extend a Quiz Due Date for an Individual Student

 

To extend the due date, start by opening the specific quiz you want to work with.
1. With the Quiz page open, go to Settings > User Overrides.
2. Click the Add User Override button.
3. Locate the students name from the User Override list. Use the search box if you have several students and cannot locate him or her easily. You may only override one student at a time.
4. Adjust the "Close the Quiz" date fields accordingly.
5. Click Save, or Save and enter another override (if you have multiple students to adjust).

 

 

 

3.   Give Additional Time to Complete a Quiz to an Individual Student

 

To provide additional time to complete a quiz, start by opening the specific quiz you want to work with.
1. With the Quiz page open, go to Settings > User Overrides.
2. Click the Add User Override button.
3. Locate the students name from the User Override list. Use the search box if you have several students and cannot locate him or her easily. You may only override one student at a time.
4. Type the new number of minutes allowed to complete the quiz in the "Time Limit" field.
5. Click Save, or Save and enter another override (if you have multiple students to adjust).

 

 

 

4.  To Allow Additional Attempts to Take a Quiz

 

 

To allow additional attempts to take a quiz, start by opening the specific quiz you want to work with. (You may or may not want to delete previous attempts as demonstrated in item #1 above, however, this is the recommended method)

1. With the Quiz page open, go to Settings > User Overrides.
2. Click the Add User Override button.
3. Locate the students name from the User Override list. Use the search box if you have several students and cannot locate him or her easily. You may only override one student at a time.
4. Set the "Attempts Allowed" field accordingly.
5. Click Save, or Save and enter another override (if you have multiple students to adjust).

NOTE: Quizzes are generally set to use the HIGHEST SCORE, if multiple attempts are given, however, you can verify this by opening and viewing the Quiz Settings page (turn editing on and then click the yellow pencil icon next to the desired quiz). Very few quizzes are set up to allow multiple attempts, so this is not a setting that is specifically checked in quality assurance testing.
