How to Zip or Compress Multiple Files into One

 

This procedure will explain how to zip or compress multiple files into one file. This is handy when turning in assignments that require multiple files, or when uploading large files.

Steps 1- 5 below are for Apple Macintosh computers.
Steps 6 - 10 are for PC Windows based machines.

 

 

 

1.  (Mac) - Locate the files in the Finder window

 

 

You'll want to save all of the files that you need to compress in the same folder, as this will make the process easier. When turning in an assignment make sure to name the files according to the instructors directions. If the instructor did not provide file naming instructions, it is a good idea to name your files in the following manner: Last name_first initial_course code_Unit #_Assignment Name. For example: Smith_B_BU101_Wk4_Resume

 

 

 

2.  (Mac) - Select (Highlight) all files to be zipped

 

 

Press the and hold Command key then click each file to select it. If you want to select all of the files in a folder, you can also click on the first file then press the Shift key and click the last folder in the list. This will select all of the files between the first and last files selected.

 

 

 

3.  (Mac) - Right click, select "Compress __ Items"

 

 

With all the files selected, right click and choose "Compress __ Items (there will be a number in the blank space representing the number of files to be compressed). This will create a new file within that folder called archive.zip.

 

 

 

4.  (Mac) - Rename the archived file

 

 

Slow double click, or press the Option key and click the archive.zip file to open the file name in edit mode. Change the file name according to the instructors directions. If the instructor did not provide file naming instructions, it is a good idea to name your files in the following manner: Last name_First initial_Course code_Unit #_Assignment Name. For example: Smith_B_BU101_Wk4_Resume

 

 

 

5.  (Mac) - Upload or turn in the zipped file

 

 

You can now submit the zipped or compressed file as your assignment. When you click the "Add Submission" button in your eLearning course, browse to the new zipped file on your computer and upload it to your course. You can also click on the zipped file and drag it to the assignment page. Drop the file into the field with the bouncing blue arrow. When the file has uploaded and appears in the field. Click the Save Changes button and follow the online instructions to submit your assignment.

 

 

 

6.  (PC) - Locate the files to zip in the Windows Explorer window

 

 

You'll want to save all of the files that you need to compress in the same folder, as this will make the process easier. When turning in an assignment make sure to name the files according to the instructors directions. If the instructor did not provide file naming instructions, it is a good idea to name your files in the following manner: Last name_first initial_course code_Unit #_Assignment Name. For example: Smith_B_BU101_Wk4_Resume

 

 

 

7.  (PC) - Select (highlight) all files to be zipped

 

 

Press and hold the Control key then click each file to select it. If you want to select all of the files in a folder, you can also click on the first file then press the Shift key and click the last folder in the list. This will select all of the files between the first and last files selected.

 

 

 

8.  (PC) - Right click, select "Send to > Compressed (zipped) File"

 

 

With all the files selected, right click and choose "Send to > Compressed (zipped) file." This will create a new file within that folder.
