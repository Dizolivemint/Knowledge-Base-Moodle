Using Spell Check in the eLearning System

 

During an update to the Pacific College eLearning system (on December 27, 2013), the spell check tool that was available in the editing tools above the textbox in discussion forums and in online text assignments (such as massage journals) was disabled because its performance was unreliable.

There is, however, a work-around.

 

 

1.  Still seeing a red dashed line below misspelled words?

 

 

You may still notice misspelled words underlined with a dashed red line as you type. This red line is actually coming from your browser (Firefox) and not the eLearning system.

 

 

 

2.  Press the Control key and right-click on the misspelled word

 

 

To view a list of suggested corrections for the misspelled word, press the control key and right-click on the word simultaneously. Select the appropriate suggestion to change the spelling of the word.

 

 

 

3.  If you do not see a red dashed line below misspelled words,

 

 

If you do not see a red dashed line below obviously misspelled words, check to make sure you are using the most recent version of Firefox. Visit www.firefox.com to download the most up-to-date version for free.
